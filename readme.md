# <u><font color="red">QR code Generator </font></u>
# <u>Information About UI:</u>
### Enter text for QR codes:
In this box you can write the multiple QR code as per your requirment. 
It will will generate the QR code line by line so make sure when you put your text information it should be line by line.
### Remark (Optional):
As name states this is an optional even if you do not write any remark still this will generate the QR code.

### Browse:
Here you can browse the location where you want to save the QR code.

### Submit:
This is action buttion which will generate the QR code to your broswe location.
